# 🍕 Pizzería Mamma Mia! - Hito 2

Este proyecto corresponde al segundo hito del curso de **React**, donde se profundiza en el manejo de **estados de los componentes** y la gestión de **eventos** mediante formularios controlados.

##  Nuevas Características (Hito 2)
- **Formulario de Registro:** Implementación del componente `Register` con validaciones de campos obligatorios, longitud de contraseña (mínimo 6 caracteres) y coincidencia entre contraseña y confirmación.
- **Formulario de Login:** Implementación del componente `Login` que valida campos obligatorios y una extensión mínima de contraseña.
- **Navegación Dinámica por Estado:** Se utiliza un estado centralizado en `App.jsx` para alternar entre las vistas de Home, Login y Register sin recargar la página.
- **Feedback al Usuario:** Uso de alertas para informar sobre el éxito o error en el envío de los formularios.

## 🛠️ Tecnologías y Conceptos Aplicados
- **React:** Uso de `useState` para el manejo de inputs controlados.
- **Eventos:** Gestión de `onChange` para capturar datos y `onSubmit` para la validación de formularios.
- **Bootstrap:** Estilizado de formularios y componentes de UI para un diseño limpio y responsivo.
- **Renderizado Condicional:** Lógica de JavaScript para mostrar componentes específicos según la interacción del usuario.
